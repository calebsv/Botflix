<h1>BotFlix🤖</h1>

<p>Olá, sejam bem-vindos ao seu sistema de streaming favorito, o BotFlix se compromete 100% com o seu humor específico para encontrar a melhor opção de filme. Você insere o seu humor ou como está se sentindo, no campo de texto e ele rapidamente acha duas opções de filme para o seu gosto, sendo rápido, fácil e prático de usar.</p>


<h1>Caracteristicas🚀:</h1>

<ol>
  <li>Interface visualmente moderna, com cores semelhantes ao Netflix</li>
  <li>Campo de texto para o humor inserido</li>
  <li>APIs que fazem a busca e mostram duas opções de filme baseado no humor</li>
  <li>Botão com efeito ripple para dar inicio ao processo</li>
</ol>

<p>O fluxo funciona da seguinte maneira: Humor inserido -> Groq(sugere filme) -> OMDb (poster, nota, sinopse) -> Groq(traduz) -> tela </p>

<h1>🛠️Techs</h1>
<ol>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>Tailwind</li>
  <li>JavaScript</li>
  <li>Netlify</li>
  <li>API Grog & OMDb</li>
</ol>
